Variáveis e Tipos Primitivos

Identificadores:

  1. Podem começar com Letras, $ ou _
  2. Não podem começar com números
  3. É possível usar Letras e Números
  4. É possível usar Acentos e Símbolos
  5. Não podem conter Espaços
  6. Não podem ser Palavras Reservadas

Dicas:

  • Maiúsculas e minúsculas fazem a diferença
  • Tente escolher nomes coerentes para suas variáveis
  • Evite se tornar um 'progamador alfabeto' (var = a,b,c,d,e...) ou um 'progrador contador' (var = n1,n2,n3,n4...)

Tipos Primitivos:

  1. Number (número): 42, 3.14, -100
  2. String (texto): 'Olá', "Mundo", 'Eu sou um texto'
  3. Boolean (lógico): true, false
  4. Null (nulo): null
  5. Undefined (indefinido): undefined
  6. Symbol (símbolo): Symbol('nome')

Exemplos

  • let idade = 30; // Number
  • let nome = 'João'; // String
  • let isAdmin = true; // Boolean
  • let pessoa = null; // Null
  • let desconhecido; // Undefined

Tipos Compostos:

  1. Object (objeto): { nome: 'João', idade: 30 }
  2. Array (vetor): [1, 2, 3], ['a', 'b', 'c']
  3. Function (função): function soma(a, b) { return a + b; }
  4. Date (data): new Date('2024-12-10')

Exemplos

  • let pessoa = { nome: 'João', idade: 30 }; // Object
  • let frutas = ['maçã', 'banana', 'laranja']; // Array
  • let soma = function(a, b) { return a + b; }; // Function
  • let dataNascimento = new Date('1990-01-01'); // Date

Tipos Especiais:

  1. NaN (não é um número): resultado de operações inválidas (ex.: 0/0)
  2. Infinity (infinito): resultado de operações com números muito grandes

Exemplos

  • let resultado = 0 / 0; // NaN
  • let infinito = 1 / 0; // Infinity

Verificando tipos

Utilize o operador typeof para verificar o tipo de uma variável:

  1. console.log(typeof 30); // "number"
  2. console.log(typeof 'Olá'); // "string"
  3. console.log(typeof true); // "boolean"
  4. console.log(typeof null); // "object"
  5. console.log(typeof undefined); // "undefined"
